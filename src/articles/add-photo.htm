When logged in, you can see the Home screen displaying active photos in the active fixlist. By default, you have no fixlists and photos. First of all you need to Working With Fixlists. Then you will be able to add photos to existing fixlists.
If you try to add a photo while you have no fixlists created, you will be notified about the need to create a fixlist first. To make sure, do the following:

  • Tap the button in the header of the Fixlist app:

  • Select the source of a photo:

  • The alert message appears:

  • Follow the steps described in the Working With Fixlists article


Add photo from the Home screen

To add a new photo to the active fixlist from the Home screen, perform the following steps:

  • Tap the button in the header of the Fixlist app:

    Select Add new photo

  • The New Photo screen appears:

  • To add a photo, tap the image placeholder. This will call the shortcut menu:

    Select the source and continue.

  • If you select Camera, take a photo, then tap Use Photo:

  • Specify the name of the photo and add description. The photo will be automatically attached to the active fixlist:

  • Save

Now the new photo is displayed on the Home screen, in the Active section.

Zooming photo

You can zoom both a photo in a fixlist and a photo attached in the chat.

Tap the button to zoom a fixlist photo, or tap the photo itself to zoom an attached photo in the chat:

Sharing and editing photos

You can move photos between fixlists, share them to Facebook, edit name and additional info, delete, report unsuitable. For this, you need to call the Photo shortcut menu:

  • Tap the button

  • To move a photo to another fixlist, select Share to fixlist

  • Select a fixlist from the list:

The Photo shortcut menu also allows you to post photos to Facebook - this is not a simple photo post: this provides link to Fixlist and lets invite people to take part in your fixlist.
