Adding participants is similar to the way described in the Create a Fixlist article. However, in this topic, the entry point is not the newly created list, but the main Fixlist menu. Follow the steps below:

  • Tap the menu button at the top of the Home screen, as shown on the screenshot below

  • The main menu appears on the left

  • Tap Participants

  • Tap the + button at the top of the My Participants screen

  • In the appeared menu select Add new participant

  • On the appeared Add new Participants screen you have 3 options:

    • Add participants by the name of an existing Fixlist user: just type in user's name - if any matches occur, the user will be shown in the autocomplete list below the input field

    • Or invite participants by e-mail: tap the @ Email button and type in the e-mail address of a person you want to invite (you can add several emails simultaneously to invite more than one person to a fixlist)

    • Or invite participants from Facebook by their usernames: tap the f facebook button and type in the Facebook username of the person you want to invite

  • When ready, tap Save

  • You will see added participants in the Participants list:

When users are added to a fixlist as participants, if they are already friends (participants) of the current user who sends tthe invitation, users will be prompted to join the fixlist. They will see the incremented counter on the Fixlist tab:

Then the invited user should switch to the fixlist he/she is invited to - and see the invitation message:

then tap the join button to join the fixlist.

In case if invited users are not previously added as participants of the current user, they won't be notified about the new invitation untill they navigate to the Participants page and add the current user to their Participants list.
