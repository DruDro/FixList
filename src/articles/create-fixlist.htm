Create a new fixlist

To create a new fixlist, follow the steps below:

  • Tap the button at the right-bottom corner of the Fixlist app:

  • The Create Fixlist screen appears:

  • Here you should Enter the name of the fixlist (the fixlist name should be unique):

  • Next you can add participants to the fixlist

    You can skip this step and add participants later. See how to add participants to an existing Fixlist in the Add Participants article

    To add participants now, perform the following steps:

    • Add participants by the name of an existing Fixlist user: just type in user's name - if any matches occur, the user will be shown in the autocomplete list below the input field:

    • Or invite participants by e-mail: tap the @ Email button and type in the e-mail address of a person you want to invite ( you can add several emails simultaneously to invite more than one person to a fixlist):

    • You will see added participants in the Participants list:

    When users are added to a fixlist as participants, if they are already friends (participants) of the current user who sends tthe invitation, users will be prompted to join the fixlist. They will see the incremented counter on the Fixlist tab:

    Then the invited user should switch to the fixlist he/she is invited to - and see the invitation message:

    then tap the join button to join the fixlist.

    In case if invited users are not previously added as participants of the current user, they won't be notified about the new invitation untill they navigate to the Participants page and add the current user to their Participants list. See the Add Participants article for more details.

  • When ready tap Create

  • The New Photo screen appears:

    You can skip the following steps and return when you are ready to add photos to the new fixlist

  • To add a photo, tap the image placeholder. This will call the shortcut menu:

    Select the source and continue.

  • If you select Camera, take a photo, then tap Use Photo:

  • Specify the name of the photo and add description. This is an optional step. You can create a photo without this data, but add it later in the edit mode. If you create a photo without a name specified, you'll see the default name: 'Photo name'

  • If you have more than one fixlist, select a fixlist from the drop-down list.

  • Next, if you tap Save, the photo will be automatically attached to the selected fixlist.

    Otherwise, you can tap the button - in this case, another fixlist will be created and the new photo will be attached to it, but not to the fixlist you created in the previous steps. As a result you will have 2 fixlists

After a new fixlist is created, photos from it are displayed on the Home screen, in the Active section.

You can switch fixlists simply by tapping the fixlist name at the top of the screen. The list of fixlists will appear. Select the needed one.

Share, edit a fixlist

The following opportunities are open for you when you have at least one fixlist:

  • Rename the fixlist

  • Share the fixlist

  • Add participants to the fixlist

  • Manage Attachments of the fixlist

  • Specify categories of the fixlist

To access all these options, perform the following steps:

  • from the Home screen, tap the Fixlist name at the top of the screen

  • Now you can see the list of your fixlists. Tap the < button next to the fixlist you want to edit or share. You will see the fixlist sub-menu

  • If you want to share the fixlist, tap the Facebook button - follow the instructions on the screen

    If you want to edit the fixlist, tap the Pencil button.

  • Here is the Edit Fixlist page. The Participants tab is open by default.

    See the Add Participants article for more details.

  • Open the Categories tab in order to assign categories to the fixlist

    Select the categories of the fixlist

  • When ready, tap Save
